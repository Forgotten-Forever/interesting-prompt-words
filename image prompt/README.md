## 🌟 魔法书目录：你的咒语分类 (Spell Categories)

我们把这些强大的视觉咒语分成了几个关键部分，让你能像一位经验丰富的炼金术士一样，精确地控制你的创作：

| 咒语类别 | 魔法效果 | 效果演示 |
| :--- | :--- | :--- |
| 🎨 **风格咒语 (Style Spells)** | 决定画面的整体艺术风格，如“蒸汽朋克”、“油画”、“赛博朋克”。 | `style: cyberpunk` 🌃 |
| 🦁 **主体召唤 (Subject Summons)** | 描述画面中的主角和核心元素，越详细越好！ | `subject: 一只戴着宇航员头盔的猫` 🐈‍⬛ |
| 📐 **构图秘籍 (Composition Secrets)** | 控制镜头的角度、景深和光线，让画面更具电影感。 | `composition: 黄金比例, 8K, 电影光` 🎬 |
| 🚫 **反向结界 (Negative Prompts)** | 告诉 AI **不要**画什么，是魔法师的“驱散术”，避免出现多余的手指或扭曲的画面。 | `negative: ugly, deformed, extra limbs` ❌ |

**记住：** 一个好的图片提示词，是**科学**与**艺术**的完美结合！ **（Prompt is Power!）** 💪
